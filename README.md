# Studio Boss

Gestor de tiempo gamificado para creativos, diseñado para equilibrar trabajo, descanso y foco durante sesiones de estudio, composición o producción.

## Objetivo del proyecto

El objetivo de esta aplicación es ayudar a personas creativas a organizar mejor su tiempo sin convertir la productividad en una cárcel con WiFi.

La app propone una forma visual y motivadora de gestionar bloques de trabajo y descanso, especialmente útil para músicos, productores, compositores, diseñadores o cualquier perfil que necesite alternar concentración profunda con pausas reales.

## Características principales

- Gestión de tiempo orientada a sesiones creativas.
- Enfoque gamificado para reforzar la constancia.
- Equilibrio entre trabajo y descanso.
- Interfaz pensada para uso diario.
- Proyecto ligero orientado a navegador.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Estado del proyecto

Proyecto funcional en fase inicial de aprendizaje y mejora continua.

## Aprendizajes trabajados

Con este proyecto se practican conceptos como:

- Organización de una aplicación web de productividad.
- Gestión de estados e interacción del usuario.
- Diseño de flujos orientados a hábitos y rutinas.
- Aplicación de gamificación a herramientas cotidianas.
- Creación de interfaces simples para casos de uso reales.
